# 建立一个webpack项目

## webpack

- npm init -y
- npm i webpack webpack-cli -D
- npm i webpack-dev-server -D
- npm i html-webpack-plugin -D
- npm i clean-webpack-plugin -D
- npm install cross-env -D
- npm i @babel/core babel-loader @babel/preset-env -D

## ts

- tsc --init
- npm i typescript
- npm i ts-loader -D
